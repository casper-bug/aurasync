💻 AuraSync Virtual Audio Driver

Install the AuraSync Virtual Audio Driver to make any Windows or macOS machine appear as an audio output device. Audio routed to this virtual device is automatically streamed to all devices in your session — turning phones, tablets and laptops into a wireless surround system.

1

Download & Install
Download the installer for your OS and run it. No reboot required.

2

Set as Default Output
In OS sound settings, set “AuraSync Virtual Out” as the default playback device.

3

Join an AuraSync Session
Any audio played through the virtual device streams to your session automatically.

4

Configure Surround Sound
Use the Surround Sound panel above to select the speaker layout for your session.

Alternatively: use VB-Cable (Windows/Mac) or PulseAudio null-sink (Linux) as a virtual audio output device.
